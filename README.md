# XGTZonesGUI

## Description


This is a PocketMine Plugin.
Open a GUI and choose where you want to teleport.

This plugin is in BETA!

Don't use commands in CONSOLE!

## How to install

1. Install plugin

2. Stop/start server

3. Edit config

4. Restart server

5. Have FUN!

## Use

This plugin use Vector3 and InvMenu by muqsit!


## Features

- [X] Config

- [X] Costom Message

- [X] Custom Item Name

- [X] Custom Type
